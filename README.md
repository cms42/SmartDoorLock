# 智能门锁

#### 项目介绍

忘记带钥匙？钥匙丢了？有了智能门锁，一切不是问题！

#### 软件架构

服务端：运行于虚谷之上，可通过WiFi及蓝牙连接，可添加指纹识别、面部识别、虹膜识别等传感器

客户端：运行于手机端
